# **FamilyCare**

## Descrição
Quando a família não tem um lugar fixo para guardar as informações do idoso, os problemas aparecem rápido. Um filho não sabe o tipo sanguíneo do pai na hora de uma emergência. Outro não lembra o telefone do cardiologista para remarcar uma consulta. O remédio das 14h foi dado duas vezes porque ninguém avisou o irmão que já tinha administrado. E o lembrete da consulta do mês que vem? Estava anotado num papel que ninguém encontra mais. São situações simples, mas que no dia a dia de uma família numerosa acontecem o tempo todo e podem colocar a saúde do idoso em risco.

## Objetivo Geral
Criar um sistema simples e acessível para que a família possa se organizar de forma colaborativa no cuidado à saúde de um idoso, reunindo em um só lugar as informações sobre remédios, consultas, médicos e o histórico de saúde do paciente.

## Objetivos Específicos
Ter um lugar centralizado com todos os dados de saúde do idoso, como alergias, tipo sanguíneo e plano de saúde;
- Guardar os contatos e informações dos médicos que o idoso frequenta;
- Registrar cada remédio administrado para que nenhum familiar aplique uma dose duplicada;
- Avisar os familiares sobre remédios, consultas e compromissos importantes do dia.
